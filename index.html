    Anthropic API Key

    Required for screenshot scanning, substitution advice, and recipe import. Stored only on this device — never sent anywhere except directly to Anthropic.

    How to get your API key

    1. Go to console.anthropic.com
    2. Sign in or create a free account
    3. Click API Keys in the left sidebar
    4. Click Create Key — name it "Kitchen Companion"
    5. Copy the key (starts with sk-ant-) and paste it above

    Tip: set a monthly spend limit in the Billing section — $5–10/month is more than enough for personal use.
